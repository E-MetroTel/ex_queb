# ExQueb

Ecto Filter Query Builder

This is a helper library for sharing code between ex_admin and auth-ex 
projects.

## Installation

Add ex_queb to your deps:

mix.exs
```elixir
  defp deps do
     ...
     {:ex_queb, "~> 0.1"}, 
     ...
  end
```

Fetch and compile the dependency

```mix do deps.get, deps.compile```
